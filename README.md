### 🚀 What I Do

I build production-grade web systems end-to-end — from database schema to API design to frontend delivery. Most of my work has been for healthcare, education, insurance, and enterprise clients, where reliability and correctness matter as much as speed.

- 🏗️ Designed and built **custom MVC frameworks from scratch** (routers, DI containers, middleware pipelines) without relying on Laravel or any other framework, when the project called for it
- 🏢 Built **multi-tenant SaaS platforms** with strict data isolation, subscription billing, and role-based access at scale
- 🔐 Designed **multi-layered authorization systems** — RBAC, policy-based access, hierarchical role structures (up to 6 levels), JWT and session-based auth
- 💳 Integrated **multiple payment gateways** (3D Secure, hosted checkout, wallet-based payment) with idempotent, transaction-safe order flows
- 🔄 Built **event-driven architectures** and **state machines** for approval workflows, appointment systems, and content lifecycles
- 🔒 Implemented **KVKK/GDPR-style data protection**: field-level encryption, hashed searchable indexes, audit logging
- 📱 Shipped **REST APIs serving both web and mobile clients** (Next.js + React Native/Expo) from a single backend
- 🐳 Containerized full-stack environments with **Docker Compose** (multi-service: app, queue, scheduler, reverse proxy, database, cache)
- 🤖 Use AI-assisted tooling actively as part of my day-to-day development workflow
